# 💄 Beauty Plus Parlour API

A scalable multi-role booking management platform for beauty parlours, built using ASP.NET Core Web API.

---

## 🚀 Tech Stack
- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- PostgreSQL
- JWT Authentication + Refresh Tokens

---

## 📦 Features
- Authentication & Role Management
- Service Management
- Staff & Customer Management
- Booking System
- Loyalty & Coupons
- Ratings & Reviews
- Notifications

---

## ⚙️ Setup

### Prerequisites
- .NET 8 SDK
- PostgreSQL

### Run
```bash
dotnet ef database update
dotnet run
